# CJK Typography — London Cuts (2026-04-28)

Drop-in CJK font system for the bilingual reader. Source of truth for the
`--f-*` family tokens, `:lang(zh)` overrides, and per-mode CJK pairings.

Generated by Claude Design (https://claude.ai/design/p/019dd121-9c50-7886-935d-364717a22672)
on 2026-04-28 against the canonical Latin set in `colors_and_type.css`. The
scale, fallback chains, and feature settings below are already wired into:

- `web/app/layout.tsx` — `GOOGLE_FONTS` array + LXGW WenKai jsDelivr link
- `web/app/globals.css` — `:root` `--f-*` cascade + section 12 `:lang(zh)` block

## 1 · Per-mode pairings

| Mode | Display CJK | Body CJK | Source | Why this pairing |
|---|---|---|---|---|
| **Fashion** | Noto Serif SC 900 | Noto Sans SC 400 | Google Fonts (OFL) | Source Han Serif Heavy is the canonical CJK didone analog — near-Mincho contrast at large sizes, mirrors Bodoni Moda's editorial weight. |
| **Punk** | Noto Sans SC 900 | Noto Sans SC 400 | Google Fonts (OFL) | Heavy hei/gothic mirrors Archivo Black's stencil weight; Source Han Sans Heavy is the 黑体 used in Chinese punk zines and protest posters. ZCOOL KuaiLe is the rougher alt (exposed via Tweaks panel in the Claude Design canvas). |
| **Cinema** | LXGW WenKai 400 | Noto Sans SC 400 | jsDelivr (lxgw-wenkai-webfont, OFL) + Google Fonts | Cinema's JetBrains Mono body becomes "subtitle"; Chinese subtitles are traditionally a clean hei or kaiti. LXGW WenKai (open source 楷体) gives the editorial italic feel that pairs with Instrument Serif italic. |

Latin characters always render in the existing Latin face (Bodoni Moda /
Archivo Black / Instrument Serif); the CJK face is reached via the cascade
when a glyph is missing in the Latin face. No JS lang-detection needed.

## 3 · `:lang(zh)` overrides — typographic tuning

Activated by `<html lang="zh-*">` (set in `app/layout.tsx` based on the
resolved locale). The font cascades above already pick the right CJK face
per token; this block only adjusts metrics.

```css
:lang(zh) {
  font-feature-settings: "palt" 1;     /* trim full-width punctuation kerning */
  text-spacing-trim: trim-start;       /* native text-spacing in Chrome/Safari */
  letter-spacing: 0;
}

:lang(zh) h1 { font-size: 76px; line-height: 1.18; letter-spacing: 0.01em; }
:lang(zh) h2 { font-size: 50px; line-height: 1.22; letter-spacing: 0.01em; }
:lang(zh) h3 { font-size: 30px; line-height: 1.35; letter-spacing: 0.01em; }
:lang(zh) p, :lang(zh) .body { line-height: 1.85; }

:lang(zh) .eyebrow,
:lang(zh) .mono,
:lang(zh) .mono-sm,
:lang(zh) .lc-eyebrow,
:lang(zh) .lc-mono,
:lang(zh) .caption {
  font-size: 12px;          /* up from 10–11px */
  letter-spacing: 0.04em;   /* down from 0.06–0.18em */
  text-transform: none;     /* CJK has no case */
}

[data-mode="fashion"]:lang(zh) h1 { font-weight: 900; letter-spacing: 0.01em; }

[data-mode="punk"]:lang(zh) h1 {
  text-transform: none;     /* override the global Punk uppercase */
  font-weight: 900;
  line-height: 1.05;
}

[data-mode="cinema"]:lang(zh) h1,
[data-mode="cinema"]:lang(zh) h2 {
  font-style: italic;       /* synthesized — LXGW WenKai ships no italic file */
  font-weight: 400;
  line-height: 1.25;
}

[data-mode="cinema"]:lang(zh) p { line-height: 1.9; letter-spacing: 0.02em; }
```

Naked-element selectors (`:lang(zh) h1`) are intentionally weak — they only
fire on truly unstyled headings. Public-page components hard-code
`fontSize: clamp(...)` inline, which wins the cascade. That's the right
behaviour: components own their layouts; the `:lang(zh)` block owns the
metrics that components don't normally set (palt, line-height, weight,
italic synthesis).

## 4 · Size scale rationale (CJK vs Latin)

| Level | Latin px | CJK px | Δ | Why |
|---|---|---|---|---|
| H1 (display) | 72 | **76** | +4 | Hanzi stroke density 1.5–2× Latin x-height; +2–4px restores visual weight. |
| H2 | 48 | **50** | +2 | Same logic, lighter bump because H2 already reads dense. |
| H3 | 28 | **30** | +2 | Stop / section titles. |
| Body | 17 | 17 | 0 | Equal px; open `line-height: 1.85` for full-width block rhythm. |
| Body short | 15 | 15 | 0 | Same. |
| Mono / caption | 11 | **12** | +1 | Below 12px CJK strokes muddy outside Retina. |
| Eyebrow | 10 | **11** | +1 | Same. |

## 5 · Quotation marks — content level (NOT yet migrated)

This is a content/i18n decision, not a CSS one. The London Cuts standard
(deferred — apply when seed content needs a quote pass):

| | English | 简体中文 |
|---|---|---|
| Comma | `,` | `，` (full-width) |
| Period | `.` | `。` |
| Quote (outer) | `"…"` | `「…」` (right-angle) |
| Quote (inner) | `'…'` | `『…』` |
| Em-dash | `—` | `——` (double, body OK) |
| Ellipsis | `…` | `……` (double) |

Why right-angle 「」 over curly "" : matches London Cuts' hard-cornered
visual grammar; modern simplified-Chinese editorial standard (used by
单读, 三联生活周刊, T 中文版); GB curly quotes "" render as ASCII
apostrophes in many fonts and visually collide with Latin runs.

Migration script (one-time) for legacy content:

```ts
// scripts/migrate-quotes.ts
const replaceQuotes = (s: string) => s
  .replace(/[“]/g, '「')   // " → 「
  .replace(/[”]/g, '」')   // " → 」
  .replace(/[‘]/g, '『')   // ' → 『
  .replace(/[’]/g, '』');  // ' → 』
```

## 6 · Verification checklist

- [ ] DevTools → Computed → `font-family` on a CJK heading shows
  Noto Serif SC (Fashion) / Noto Sans SC (Punk) / LXGW WenKai (Cinema),
  not PingFang SC or system fallback.
- [ ] Network panel: CJK font subsets stream lazily as encountered
  (filename pattern `notosanssc-...subset-NN.woff2` etc.). Initial CJK
  payload < ~280 KB.
- [ ] Mode switch (Fashion → Punk → Cinema) re-renders CJK heading
  cleanly with no FOUT/FOIT flash.
- [ ] `「」` quotes render as right-angle brackets, not boxes (◻).
- [ ] Firefox: `font-feature-settings: "palt" 1` does the kerning
  (Firefox doesn't yet support `text-spacing-trim`).

## 7 · Budget

- **Initial CJK payload (zh page first paint):** ~280 KB (Noto Sans SC 400 only — Google's `unicode-range` chunking serves the most common ~1000 hanzi first; rest stream as encountered).
- **Total CJK after warm cache (all 3 modes, full reading session):** ~1.3 MB.
- **English-only sessions:** zero CJK download (unicode-range cascade never triggers).
